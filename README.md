# Morning Digest

## Description

Morning Digest is a full-stack news application designed to deliver personalized news digests to users via email subscriptions. The backend manages user authentication, news storage, and email notifications, while the frontend offers an intuitive interface for user interaction and subscription management.

## Technologies Used

- **Backend:**
  - Node.js
  - Express.js
  - Postgres
  - bcrypt (password hashing)
  - jsonwebtoken (authentication)
  - Resend (email sending)
  - ejs (email templating)
  - zod (data validation)
  - cors (cross-origin resource sharing)

- **Frontend:**
  - HTML
  - CSS
  - JavaScript

## Installation

### Backend

1. Clone the repository.
2. Navigate to the `App de Noticias Backend` directory.
3. Install dependencies: `npm install`.
4. Set up a Postgres database and configure connection details in environment variables.
5. Set environment variables (e.g., `PORT`, database credentials, JWT secret, Resend API_KEY).
6. Run the development server: `npm run dev`.

### Frontend

1. Clone the repository.
2. Navigate to the `App de Noticias` directory.
3. Install dependencies: `npm install`.
4. Set up your backend in main.js.
5. Run the development server: `npm run dev`.

## Usage

- Users can register and log in to manage their news subscriptions.
- The application sends personalized news digests via email based on user preferences.
